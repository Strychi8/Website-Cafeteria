# ☕ Baristas - Landing Page de Cafetería

Una landing page moderna y totalmente responsiva diseñada para una cafetería, desarrollada con **HTML5** y **CSS3**. Este proyecto demuestra el uso de técnicas modernas de maquetación web y diseño visual atractivo.

## 🚀 Tecnologías Utilizadas

- **HTML5:** Estructura semántica del contenido.
- **CSS3:** Estilos avanzados incluyendo:
  - **Flexbox & CSS Grid:** Para la distribución de elementos complejos y la galería de imágenes.
  - **Variables CSS:** Para mantener una paleta de colores consistente.
  - **Media Queries:** Para adaptar el diseño a dispositivos móviles y tablets.
  - **Pseudo-elementos y Hover Effects:** Interactividad visual.
- **FontAwesome:** Iconos para la interfaz de usuario.
- **Google Fonts:** Tipografía "Poppins" para un estilo moderno.

## 📂 Características de las Secciones

1.  **Header:** Barra de navegación, búsqueda y acceso rápido a carrito/usuario.
2.  **Hero Section:** Banner principal atractivo.
3.  **Características:** Iconos informativos (envío, soporte, etc.).
4.  **Categorías:** Destacados visuales de tipos de café.
5.  **Productos:** Grid de productos con precios, descuentos y valoraciones.
6.  **Galería:** Composición visual usando CSS Grid.
7.  **Blog:** Últimas noticias y artículos.
8.  **Footer:** Información de contacto, enlaces y newsletter.

## 💻 Cómo visualizar el proyecto

Simplemente abre el archivo `index.html` en tu navegador web preferido. No se requiere instalación ni servidor backend.
